6 自定义变量

  • 描述 自定义变量。目前只支持5个app级别变量(scope参数为visit)，索引从1-5
    发布路径 - (BOOL)setCustomVariableForIndex:(NSUInteger)index
    name:(NSString*)name value:(NSString*)value scope:( NSString*)scope;
    调用说明 把一个name-value对放在app级别（随着app销毁而销毁），以至于把调用该函数
    后的埋点数据归因到name-value对
    输入参数 参数变量 参数名称 类型 参数说明 必须
    索引 Index NSUInteger 1-5 是
    字名 Name String 是
    值 Value String 否
    范围 Scope String 当前值仅为visit 是
