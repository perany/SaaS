4.2 自定义事件

  • 调用接口：void trackEvent(String path, String urlRef, String category, String action, Map <String, String> customData)
  • 参数名 类型 说明
    path 字符串 当前页面路径，允许空值
    urlRef 字符串 前一个页面路径，允许空值
    category(分类) 字符串 固定值click
    action(行为) 字符串 SAAS平台管理中的事件code
    customData Map<String,String> 自定义参数，允许null
  • trackEvent用于跟踪页面的行为交互。
