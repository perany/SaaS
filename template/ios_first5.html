1.5 添加编译参数

  • 1、添加Linking参数：
    • 1）找到Linking设置
    • 2）在Other Linker Flags 中添加 -ObjC 参数，请注意大小写
    • 提醒：
      Linking 设置位于 项目设置 target -> 选项卡 Build Settings，左上角选择 All。
  • 2、添加Macro参数：
    • 1）添加FONOVA_ANALYTICS_IDFA=1
    • 用于获取IDFA，但根据 App Store 的审核条款，打开该选项采集 IDFA 却未集成任何广告服务的 App，可能会在审核时受阻。
    • 提醒：
      Macro 设置位于 项目设置 target -> 选项卡 Build Settings，左上角选择 All。
