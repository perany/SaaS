4.5 自定义变量

  • 调用接口：void setCustomVariable(int index, String name, String value, String scope)
  • 参数名 类型 说明
    Index 整型 目前只支持5个自定义变量，索引从1-5。
    可重复针对同一个索引调用该函数，新值
    覆盖旧值。
    Name 字符串 变量名
    Value 字符串 变量值，可空
    Scope 字符串 当前值仅为visit
  • setCustomVariable把一个name-value对放在app级别（随着app销毁而销毁），以至于把调用该函数后的埋点数据归因到name-value对。
  • 示例代码
  • setCustomVariable(1,“campaignName”,“SpringSales”,“visit”)
