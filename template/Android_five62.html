4.6.2 其他来源监测

  • 应用被安装之后，即可被来自广告系列、网站或其他应用的引荐所启动，启动时需携带referrer参数。需在MainActivity的onCreate中调用setCampaignParams，参数值即为referrer参数值。
  • 第三方app启动示例：可在第三方app启动第一方app的代码中增加Intent参数referrer，并在第一方MainActivity的onCreate中调用void setCampaignParams(String params),params即为intent参数referrer值。
  • 代码示例
  • 第三方：
  • Intent intent = new Intent(Intent.ACTION_MAIN);
  • intent.putExtra("referrer","utm_source=test_source&utm_medium=test_medium&utm_term=test_term & utm_content=test_content&utm_campaign=test_name");
  • intent.setComponent(new ComponentName("com.package.address","com.package.address.MainActivity"));
  • startActivity(intent);
  • 第一方：
  • String referrer = this.getIntent().getStringExtra("referrer");
  • if(referrer != null){
  • getTracker().setCampaignParams(referrer); }
  • H5启动示例：referrer参数值取自Sass平台渠道URL 中utm_source参数部分， 然后调用setCampaignParams。
